GREP COMMAND CHEATSHEET

01. BASIC MATCHING

COMMANDDESCRIPTIONEXAMPLE
grep "str" fileSearch for string in filegrep "error" log.txt
grep -iCase insensitive searchgrep -i "User" log.txt
grep -vInvert match (exclude string)grep -v "success" log.txt
grep -wMatch whole words onlygrep -w "is" file.txt

02. RECURSIVE & FILE SCOPE

COMMANDDESCRIPTIONEXAMPLE
grep -rRecursive search in directorygrep -r "config" /etc/
grep -lShow filenames ONLY (with match)grep -l "error" *.log
grep -LShow filenames ONLY (no match)grep -L "done" *.log
grep --excludeExclude specific filesgrep "foo" . --exclude=*.git

03. OUTPUT CONTROL

COMMANDDESCRIPTIONEXAMPLE
grep -nShow line numbersgrep -n "TODO" main.c
grep -cCount matches (per file)grep -c "error" log.txt
grep -oPrint ONLY matching partgrep -o "v[0-9.]+" file
grep --colorHighlight matches (default often)grep --color "fail" log

04. CONTEXT CONTROL (A/B/C)

VISUALIZING CONTEXT FLAGS: ========================== LINE 1: ... (ignored) LINE 2: ... (ignored) LINE 3: PRE-CONTEXT <--- -B 2 (Before) LINE 4: PRE-CONTEXT <--- LINE 5: >> MATCH << <--- "grep pattern" LINE 6: POST-CONTEXT <--- LINE 7: POST-CONTEXT <--- -A 3 (After) LINE 8: POST-CONTEXT <--- NOTE: -C 2 (Context) equals -A 2 AND -B 2
COMMANDDESCRIPTIONEXAMPLE
grep -A <n>Print n lines AFTER matchgrep -A 5 "Error" log
grep -B <n>Print n lines BEFORE matchgrep -B 2 "Error" log
grep -C <n>Print n lines context (Before+After)grep -C 3 "Error" log

05. ADVANCED & REGEX

COMMANDDESCRIPTIONEXAMPLE
grep -EExtended Regex (like egrep)grep -E "err|warn" log
grep -FFixed strings (fast, no regex)grep -F ".*" file
grep ^Anchor: Start of linegrep "^sudo" auth.log
grep $Anchor: End of linegrep "failed$" auth.log
grep -f fileGet patterns from a filegrep -f pattern.txt log
