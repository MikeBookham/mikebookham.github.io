WINDOWS KEYBOARD SHORTCUTS DATABASE

01. ESSENTIAL SYSTEM SHORTCUTS

KEY COMBOFUNCTION
WIN + LLock your PC instantly
WIN + DShow/Hide Desktop (minimize all)
WIN + EOpen File Explorer
WIN + ROpen Run dialog box
WIN + XOpen Quick Link menu (Power User Menu)
WIN + IOpen Windows Settings
WIN + S / QOpen Search
WIN + . (Period)Open Emoji Panel / Clipboard History
CTRL + SHIFT + ESCOpen Task Manager directly

02. WINDOW MANAGEMENT, MOVING & RESIZING

KEY COMBOFUNCTION
WIN + LEFT / RIGHTSnap window to left/right half
WIN + UPMaximize window
WIN + DOWNMinimize or restore window
ALT + SPACE, then MMOVE: Use Arrow Keys to move, Enter to drop
ALT + SPACE, then SRESIZE: Use Arrow Keys to resize, Enter to set
WIN + SHIFT + ARROWMove window to another monitor
WIN + TABOpen Task View (Timeline / Virtual Desktops)
ALT + TABSwitch between open apps

03. VIRTUAL DESKTOPS

KEY COMBOFUNCTION
WIN + CTRL + DCreate a new virtual desktop
WIN + CTRL + LEFT/RIGHTSwitch between virtual desktops
WIN + CTRL + F4Close current virtual desktop

04. FILE EXPLORER COMMANDS

KEY COMBOFUNCTION
ALT + UPGo up one directory level
ALT + LEFT / RIGHTGo back / forward in history
F2Rename selected file
CTRL + SHIFT + NCreate new folder
ALT + ENTEROpen Properties for selected item
SHIFT + DELETEDelete permanently (skip Recycle Bin)

05. SCREENSHOTS & RECORDING

KEY COMBOFUNCTION
WIN + SHIFT + SSnipping Tool (Region screenshot)
WIN + PRT SCNSave full screenshot to Pictures folder
WIN + GOpen Game Bar (Recording/Audio)
WIN + ALT + RStart/Stop Screen Recording

06. TEXT & CLIPBOARD

KEY COMBOFUNCTION
WIN + VOpen Clipboard History (Must enable first)
CTRL + ASelect All
CTRL + C / X / VCopy / Cut / Paste
CTRL + Z / YUndo / Redo
