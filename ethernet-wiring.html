04. NOTES

  • ELECTRICAL: There is NO electrical performance difference between A and B. Both work identically.
  • CONSISTENCY: The most important rule is to pick one standard and use it on BOTH ends of the cable.
  • AUTO-MDIX: Modern switches (Gigabit+) detect the cable type automatically, making Crossover cables largely obsolete.
  • POE: Power over Ethernet uses the same pins but injects DC voltage. Standard wiring is required for safe operation.
